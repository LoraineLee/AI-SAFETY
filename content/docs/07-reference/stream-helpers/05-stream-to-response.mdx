# `streamToResponse`

This method will pipe an AI stream to a Node.js `ServerResponse` object.
It can be helpful to combine this with other AI stream utilities, such as OpenAIStream, in Node.js environments.
Similar to StreamingTextResponse, it automatically sets the status code to 200 and the Content-Type header to 'text/plain; charset=utf-8'
