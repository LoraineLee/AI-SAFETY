# Streaming Values

The RSC API allows you to stream serializable JS values from the server to the client using [`createStreamableValue`](/docs/reference/ai-sdk-rsc/create-streamable-value), such as strings, numbers, objects, and arrays.

This is useful when you want to stream:

- Text generations from the language model in real-time.
- Buffer values of image and audio generations from multi-modal models.
- Progress updates from multi-step agent runs.

## Creating a Streamable Value

You can import `createStreamableValue` from `ai/rsc` and use it to create a streamable value. When working with RSCs, you can use it alongside any of the SDK's functions for generating responses from a model provider.

```tsx file='app/actions.ts'
'use server';

import { openai } from '@ai-sdk/openai';
import { createStreamableValue } from 'ai/rsc';
import { streamText } from 'ai';

export const generateBedtimeStory = async (input: string) => {
  const stream = createStreamableValue<string>();

  (async () => {
    const result = await streamText({
      model: openai('gpt-3.5-turbo'),
      prompt:
        'Write a bedtime story about open source software.',
    });

    for await (const partialStream of result.textStream) {
      stream.append(partialStream)
    };

    stream.done();
  })();

  return {
    value: stream.value,
  };
};
```

## Reading a Streamable Value

You can read streamable values on the client using `readStreamableValue`. It returns an async iterator that yields the value of the streamable as it is updated:

```tsx file='app/page.tsx'
'use client'

import { readStreamableValue } from 'ai/rsc';
import { generateBedtimeStory } from '@/actions';

export default function Page() {
  const [result, setResult] = useState('')

  return (
    <>
      <button
        onClick={async () => {
          const { value } = await generateBedtimeStory();

          for await (const partialValue of readStreamableValue(value)) {
            if (partialValue) setResult(partialValue);
          }
        }}
      >
        Generate Bedtime Story
      </button>
      <p>{result}</p>
    </>
  );
}
```
